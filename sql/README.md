## SQL Files Documentation

### explore_tables.sql
This file contains SQL queries that are used to explore various tables in the database. It focuses on understanding the data structure and the content of each table, allowing users to identify key relationships and insights.

### business_questions.sql
This file is designed to provide answers to the business questions raised during the project. It includes SQL queries that synthesize the data from different tables to derive actionable insights that can inform decision-making processes.
